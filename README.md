    </ul>

    </p>

    <h2>2º Passo:</h2>
    <p>Selecione a plataforma desejada para obter os dados (Udemy, Kiwify ou nutror).</p>

    <p>Após esses passos, a extensão estará pronta para baixar os conteúdos desejados!</p>
